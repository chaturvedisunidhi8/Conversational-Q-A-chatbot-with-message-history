# Conversational-Q-A-chatbot-with-message-history

This project is an intelligent conversational chatbot that can answer questions and remember past interactions! By storing message history, the chatbot provides context-aware responses, making conversations more natural and engaging. Ideal for customer support, learning assistants, or personal virtual companions.

Features:

🧠 Contextual Q&A: Answers based on current and previous messages.

📚 Message History: Keeps track of conversations for better context.

🌐 Web/Knowledge Integration: Pulls information from websites or documents.

🤖 Interactive Interface: Can be used with Streamlit, web apps, or other platforms.

🛠️ Extensible: Add more tools, APIs, or custom logic easily.

Technologies Used:

Python 🐍

LangChain 🔗

Ollama/ LLMs 🤖🗄️
